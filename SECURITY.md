# Política de Seguridad

| Versión | Soportada |
| ------- | --------- |
| Main    | ✅        |

## Reportar una Vulnerabilidad

Si descubres una vulnerabilidad de seguridad en este proyecto, por favor no abras un issue público. En su lugar, contacta con nosotros de forma privada a través de los datos de contacto del perfil o el correo asociado al repositorio.

Nos comprometemos a revisar y responder a todos los reportes de seguridad de forma diligente.
